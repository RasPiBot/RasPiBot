À propos

Une plateforme de robotique open source accessible aux débutants et aux experts.
Apprenez la programmation, l'électronique, Linux, la réseautique et l'impression 3D.

Faible coût

Nous vous apprendrons à construire votre robot humanoïde pour moins de 200$.

Raspberry Pi

Le contrôleur Raspberry Pi est un ordinateur complet. Votre robot aura accès à Internet.

Coque 3D imprimable

Tous nos modèles sont open source et peuvent être fabriqués avec une imprimante 3D.

Le contrôleur du robot est un Rasberry Pi jumelé à un chapeau Adafruit 16 Channel Servo Driver.

Raspberry Pi 3
  • 1.2GHz 64-bit quad-core ARMv8 CPU
  • 1GB RAM
  • 40 GPIO pins
  • Full HDMI port
  • 802.11n Wireless LAN
  • Bluetooth 4.1
  • Fast-Ethernet port
  • Micro SD card slot
Adafruit 16 Channel Servo Driver
  • Contrôle jusqu'à 16 moteurs.
  • Empiler une autre carte pour 16 autres moteurs.

Mission

Nous croyons que TOUS les enfants devraient pouvoir s'offrir un robot humanoïde. RasPiBot offre un kit de robotique abordable avec une mentalité "open source". Découvrez la puissance du RaspBerry Pi et commencez à programmer votre robot!

Librairie Open source

Notre librairie Python fonctionne avec tous les environnements de développement.

Modèles 3D Open source

Imprimez et modifiez les coques des robots.

Pièces abordables

Nous avons déniché les pièces les plus abordables pour que tout le monde saute dans l'univers de la robotique.

Écran touch screen

Contrôlez votre robot à l'aide de l'écran situé sur le ventre.

Caméra

Utilisez la reconnaissance faciale avec la caméra HD.

Partage avec la communauté

Partagez votre code avec notre communauté sur GitHub et vos modèles 3D sur Thingiverse.

Sans-fil

Contrôlez votre robot à distance à l'aide de SSH.

Bluetooth

Entrez dans le monde de la domotique à la maison. Votre robot peut tout contrôler.

4 ports USB

Connectez n'importe quel périphérique USB à votre robot. Votre robot est un ordinateur complet.
